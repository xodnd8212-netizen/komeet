# 어드민 계정 정보

## 어드민 계정 생성 방법

### 방법 1: 앱에서 직접 생성 (가장 간단)

1. 앱을 실행합니다
2. 로그인 페이지에서 **"회원가입"**을 선택합니다
3. 원하는 이메일과 비밀번호로 계정을 생성합니다
4. 로그인 후 프로필 페이지로 이동합니다
5. **어드민 대시보드에 접근**하려면:
   - URL을 `/admin`으로 변경하거나
   - 어드민 대시보드에서 "현재 계정을 어드민으로 승격" 버튼을 클릭합니다
6. 어드민 대시보드에서 **"현재 계정을 어드민으로 승격"** 버튼을 클릭합니다
7. 확인 대화상자에서 "승격"을 선택합니다
8. 이제 해당 계정으로 로그인하면 자동으로 어드민 대시보드로 이동합니다

### 방법 2: Firebase 콘솔에서 수동 생성

1. Firebase 콘솔 → Authentication → 사용자 추가
2. 이메일과 비밀번호를 입력하여 계정 생성
3. 생성된 사용자의 UID를 복사합니다
4. Firestore → `admins` 컬렉션 → 새 문서 추가 (문서 ID는 사용자 UID)
5. 다음 JSON 데이터를 추가합니다:

```json
{
  "isAdmin": true,
  "email": "사용자 이메일",
  "adminName": "시스템 관리자",
  "createdAt": "서버 타임스탬프",
  "updatedAt": "서버 타임스탬프"
}
```

## 사용 방법

1. 어드민 계정으로 로그인합니다
2. 자동으로 어드민 대시보드로 이동합니다
3. 어드민 대시보드에서 모든 관리 기능을 사용할 수 있습니다

## 어드민 권한

어드민 계정으로 로그인하면 다음 기능을 사용할 수 있습니다:

- ✅ 모든 사용자 프로필 조회 및 관리
- ✅ 모든 매칭 조회 및 관리
- ✅ 모든 채팅방 조회 및 관리
- ✅ 결제 내역 조회 및 관리
- ✅ 코인 거래 내역 조회 및 관리
- ✅ 사용자 계정 삭제
- ✅ 전체 앱 통계 조회
