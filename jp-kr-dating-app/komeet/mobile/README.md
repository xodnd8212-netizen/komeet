# Komeet Mobile Dating App

Welcome to the Komeet mobile dating app project! This application connects users for dating and social interactions, featuring a user-friendly interface inspired by popular dating apps like Wippy, Glam, and Tinder.

## Project Structure

The project is organized into several key directories:

- **lib/**: Contains the main application code, including UI components, services, and models.
- **android/**: Contains Android-specific configurations and files.
- **ios/**: Contains iOS-specific configurations and files.
- **test/**: Contains widget tests for the application.
- **services/**: Contains services for API calls, authentication, and file uploads.
- **features/**: Contains feature-specific components such as profile, match, and settings pages.
- **theme/**: Contains theme-related files for consistent styling across the app.
- **utils/**: Contains utility functions for validation and other common tasks.

## Getting Started

To get started with the Komeet mobile app, follow these steps:

1. **Clone the repository**:
   ```
   git clone <repository-url>
   cd komeet/mobile
   ```

2. **Install dependencies**:
   ```
   flutter pub get
   ```

3. **Run the app**:
   ```
   flutter run
   ```

## Features

- User profiles with customizable settings.
- Swipe functionality for matching with other users.
- Bottom navigation bar for easy access to different sections of the app.
- Push notifications for real-time updates.
- Secure authentication and user data management.
