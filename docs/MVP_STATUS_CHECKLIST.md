## 우선순위 액션 (vNext)
1. **인증 확장**: Apple Sign-In, Phone OTP, 익명→정식 계정 업그레이드 플로우 설계 및 구현.  
2. **모더레이션 파이프라인**: 사진/텍스트 분석 Functions, 점수 누적/제재 로직, 신고/차단 데이터 모델 정의.  
3. **매칭·채팅 SLA**: 추천/매칭/채팅 latency 측정, 타이핑/신고 버튼 추가, Functions 기반 푸시.  
4. **운영 콘솔**: 신고 큐·사용자 상태 변경·감사로그 요구에 맞춘 최소 기능 어드민 앱 설계.  
5. **정책/콘텐츠**: 커뮤니티 가이드/약관 초안 작성, 앱 내 정책 뷰 및 19+ 배지 노출, 권한 안내 UX 제작.

작업 진행 시 본 문서를 주기적으로 업데이트하여 남은 요구 사항을 추적하세요.
